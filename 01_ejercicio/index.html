CONCEPTO DE GIT

Git is the open source distributed version control system that facilitates GitHub activities on your laptop or desktop. This cheat sheet summarizes commonly used Git command line instructions for quick reference.

comandos principales de git

  • git init: inicialisar un proyecto en git.
  • git config --global user.email (email): identificar y configurar globalmente el email en un repositorio local
  • git config --global user.name (name): identificar y configurar globalmente el nombre de usuario en un repositorio local
  • git config user.name: se usa para identificar el nombre de usuario en el repositorio local
  • git config user.email: se usa para verificar el email en el repositorio local
  • git config --global --unset-all user.name: elimina las configuraciones del nombre de usuario en el repositorio local
  • clear: borrar la pantalla del terminal
  • cd (nombre del directorio): ingresa a un directorio en la terminal
  • cd ..: salir del directorio en la terminal
  • pwd: mostrar la ruta del directorio actual en la terminal
  • ls: muestra la lista de contenidos de un directorio
  • git log: mostrar el historial de cambios en el repositorio
  • git add .: agrega todos los archivos modificados al repositorio
  • git status: verificar el estado del repositorio
  • git commit -m "(comentario)": guarda los cambios realizados en el repositorio con un mensaje resumido
  • git branchmostrar el nombre de la rama
  • git branch -m (nombre de la rama): esto nos permite cambiar e nombre de la rama
